Class: World

CrunchJS. World

new World(config)

This is the main object for the CrunchJS Engine. It can be thought of as a collection of scenes, with each scene will have its own data and entities. Scenes can be added, removed, and transitioned between. This creates a new instance of the World CrunchJS and should be a singleton. Global access at CrunchJS.world.
Parameters:
Name Type Argument Description
config Object | string <optional>
 The configurations for the engine.
Properties
Name Type Argument Description
fps Number <optional>
 The frames per second to run at. Usually leave this undefined so that the engine can pick the optimal rate.

// Create a world that uses a webworker for the simulation
var world = new CrunchJS.World();
