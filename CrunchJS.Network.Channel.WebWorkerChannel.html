CrunchJS.Network.Channel. WebWorkerChannel

Communicates between Web Workers and Main windows

Creates a Web Worker Channel
Parameters:
Name Type Argument Description
worker Worker <optional>
 The worker to talk to. If the worker is undefined, the channel will assume that the engine is being run in a web worker, and will connect to the postmessage and onmessage that way.
